## 绩点和果酱

很久之前（可能大约是初中），我就有过这样的想法：一个阶段结束，重要的不是取得了什么成果，而是要理解这个成果是怎么来的，为什么你可以拿到成果。我们经常地会过高估计成果的主观因素，认为这都是自己努力争取获得的，从而认为一切都是理所当然的；而事实却是，**破天荒的运气**、**贵人的相助**、**掌握的信息差**或者是**平台的力量**将你推举到了那样的高度，而自己所做的，无非是站在梯子上伸手，就摘下了树上的苹果。

因此站在大二的节点上会看大一，拿下果酱的因素里，相当大的一部分因素应当是几乎所有的数学基础课（高数线代概率论）都在高中物竞阶段有了系统性的学习，并且以极高的强度和难度进行训练，这部分长期刷题所获得的记忆，再经过稍微考前的复习唤醒，便能够在难度很高的高数下和概率论考试中稳定发挥，以很高的学分取得优势。

另外一部分因素应该是运气，特别是选修课没有选到雷课，老师给分也比较善良，再加上考勤和作业不是完全摆烂的，选修这部分应该有不少的优势。

剩下的一部分因素或许是自己确实比较专注，平日不打游戏，也没有NPY，在大一的时候对西安的人文景点也并不感兴趣，并且也不急着做科研，因此主要的精力完全在课内学习上，投入的时间比较多。（现在看起来过于多了，完全没有必要，不过当时也没有其他可做的事情）

不过到大二，从卷绩点这件事上看，情况发生了一些变化。首先数学基础课除了复变之外已经没有了，所有课都是专业课；这使得我大一先学习的优势完全没有了。其次，科研的任务变成了生活另一条主线，要做出来至少合格的工作，总是要投入相当大量的一段时间的，这使得我充分学习了 axi 的讲座方法，一些 2 学分的课直到考前 2-3 天才开始预习。另外，运气这件事并非恒定，总是有好有坏。

总之，大一的成绩应当只是一个结果，或许是对高考不利的一种宽慰，但无论如何，大二的我是一个新的我，大二的成绩也完全不可能和大一一样。

**果酱是一个标志**，证明一个人在应试能力上的突出，从而间接反应这个人踏实、勤奋、稳定、心理素质良好等一系列品质。一个果酱足以证明这些素质，两个三个果酱除了挂在简历上吹吹，也很难有其他作用，因为它很难反映出一个人真实的创新能力、项目能力、管理协调能力等等。
